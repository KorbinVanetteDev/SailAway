  OCEAN TRADING ADVENTURE


VERSION: 1.0
ENGINE: Unity 2022.3.62f3

ABOUT THE GAME

Sail the open seas, explore mysterious islands, collect valuable
resources, and build your trading empire! Start with a basic ship
and work your way up to commanding a legendary pirate galleon.

HOW TO PLAY


1. Launch the game
2. Click "Play" to continue or start new game
3. Use WASD to sail your ship
4. Visit islands to trade resources
5. Collect wood, iron, and cloth
6. Earn coins to unlock new ships
7. Explore all three islands
8. Build your fleet!

CONTROLS

W / Up Arrow       - Move Forward
S / Down Arrow     - Move Backward
A / Left Arrow     - Turn Left
D / Right Arrow    - Turn Right
M                  - Toggle Minimap
ESC                - Pause Menu


FEATURES

- Open ocean sailing
- Three explorable islands
- Resource trading system
- Multiple unlockable ships
- Pirate ship with cannons
- Real-time minimap with compass
- Automatic save system
- Continue game feature
- Save code backup system


SYSTEM REQUIREMENTS


MINIMUM:
  OS: Windows 7/8/10/11 (64-bit)
  Processor: Dual-core 2.0 GHz
  Memory: 4 GB RAM
  Graphics: DirectX 11 compatible GPU
  Storage: 500 MB available space

RECOMMENDED:
  OS: Windows 10/11 (64-bit)
  Processor: Quad-core 2.5 GHz
  Memory: 8 GB RAM
  Graphics: DirectX 11 compatible GPU with 2GB VRAM
  Storage: 1 GB available space


INSTALLATION


1. Extract the ZIP file to desired location
2. Run SetSail.exe (or your game name)
3. Configure graphics settings in launcher
4. Click "Play!"

IMPORTANT: Please don't separate the .exe file from the _Data folder!




TIPS & TRICKS

- Visit all three islands to find the best trades
- Save up coins to buy supplies from the shop when they are in stock.
- Use the minimap to navigate between islands
- Your progress saves automatically when you quit through the menu
- Copy your save code before starting a new game
